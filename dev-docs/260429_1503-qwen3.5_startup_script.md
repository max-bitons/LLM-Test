# Qwen 3.5 啟動程式說明

> created by : bitons & cursor

## 簡介
參考了原先的 `start_vllm_server_llama_31b.sh`，我們建立了一支專用於啟動 Qwen 3.5 模型的 vLLM 伺服器腳本：`start_vllm_server_qwen3.5.sh`。

## 主要修改項目
1. **預設模型更改**：將預設模型更改為 `Qwen/Qwen3.5-9B`，並提供 `QWEN_MODEL_ID` 變數供覆寫。
2. **通訊埠更改**：將預設的 API Port 從 8001 改為 `8002` (`VLLM_API_PORT=8002`)，避免與 Llama 伺服器在同一台機器上執行時發生通訊埠衝突。
3. **量化與資料型態設定**：
   - 移除了原先寫死的 NVFP4 設定，將 `VLLM_DTYPE` 設為 `auto`。
   - 將 `VLLM_QUANTIZATION` 預設為空值，若有需求可透過環境變數指定為 `awq`、`gptq` 等量化格式。
4. **環境變數調整**：動態處理量化參數 (`--quantization`)，只有在有設定 `VLLM_QUANTIZATION` 時才會將該參數傳遞給 vLLM。

## 使用方式
直接執行腳本：
```bash
./start_vllm_server_qwen3.5.sh
```

若要使用自訂模型或量化格式，可透過環境變數覆寫：
```bash
QWEN_MODEL_ID=Qwen/Qwen3.5-9B-AWQ VLLM_QUANTIZATION=awq ./start_vllm_server_qwen3.5.sh
```
